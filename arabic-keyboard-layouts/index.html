The arabic keyboard

The arabic keyboard is the keyboard used to type arabic alphabets.It contains both latin and arabic because latin letters and numbers are used for different things which can not be typed in arabic such as emails or urls.

Arabic keyboard layouts

A layout is simply how characters are distributed relatively to each other.Different layouts are used by companies which make keyboards .So lets see the diffrent layouts used by arabic keyboards.
